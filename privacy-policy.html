Política de Privacidade

CortesTop — Última atualização: 28 de abril de 2026

1. Informações que coletamos

O aplicativo CortesTop não coleta dados pessoais de usuários finais. O aplicativo é utilizado exclusivamente para publicação automática de conteúdo de vídeo em plataformas de redes sociais (Facebook, Instagram, TikTok e YouTube), operado pelo proprietário da página.

2. Uso das informações

As únicas informações acessadas pelo aplicativo são as credenciais de acesso às APIs das plataformas (tokens de acesso), utilizadas exclusivamente para realizar uploads de vídeo e publicações em nome do proprietário da página. Essas credenciais não são compartilhadas com terceiros.

3. Compartilhamento de dados

Não compartilhamos nenhum dado com terceiros. O aplicativo não vende, aluga ou divulga informações pessoais a nenhuma parte externa.

4. Armazenamento de dados

Os tokens de acesso são armazenados localmente no servidor do proprietário da página e utilizados apenas para operações de publicação. Não há banco de dados de usuários.

5. Direitos do usuário

Como o aplicativo não coleta dados de usuários finais, não há dados pessoais para solicitar exclusão ou acesso. Caso tenha dúvidas, entre em contato pelo e-mail abaixo.

6. Alterações nesta política

Esta política pode ser atualizada periodicamente. A data de "última atualização" no topo desta página indica quando ocorreu a revisão mais recente.
